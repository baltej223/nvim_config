# Neovim Config

## Requirements

- Neovim
- git
- ripgrep
- fd

## Installation

```
git clone https://github.com/baltej223/nvim_config ~/.config/nvim
```

or ssh one, if you prefer that,

```
git clone git@github.com:baltej223/nvim_config.git ~/.config/nvim
```

## Keybindings

- `<leader>` -> space key
- `<leader><leader>` -> find files
- `<leader>bd` -> Close Current Buffer

## Plugins

- lazy.nvim
- telescope.nvim
- treesitter
